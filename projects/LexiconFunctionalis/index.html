Lexicon Functionalis

Technologies: Haskell

Purpose

The goal of this project is to utilize the principles and foundations of functional programming to create a network connected, online dictionary written in the Haskell language.

The user should be able to connect to dict.org (port 2828) to access the multiple dictionaries available there, some of them being in different languages.

Design

The project employs some of the basic principles of networking such as using the library, HaskellNet, to send requests to and recieve responses from dict.org. The project in its current iteration consists of a CUI instead of a GUI, which will be implemented at a later time.
