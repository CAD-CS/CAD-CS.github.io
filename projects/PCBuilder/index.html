PC Builder

Technologies: PostgreSQL, JavaScript, NodeJS, React

Purpose

Building your own PC is no easy feat, so why not try to make it easier?

The purpose of this project is to provide an easy to use application that empowers users to mix, match and compare different PC components. The user can see the individual component's performance and other relevant information such as year, make, and manufacturer.

This allows the user to experiment with different build and to determine which is best for their purposes.

Design

From a technical perspective, the application uses PostgreSQL as its SQL dialect along with JavaScript and NodeJS. The frontend UI is made using React.
